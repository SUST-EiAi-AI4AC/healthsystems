2. 切换到 Console 标签
3. 导入并运行测试脚本：

```javascript

// 或者直接复制函数到控制台运行
```

## 🐛 常见问题排查

### 问题 1：点击历史记录按钮无反应
**检查项：**
- 确认页面已完全加载
- 检查浏览器控制台是否有错误
- 确认 uni API 可用

### 问题 2：历史列表为空
**检查项：**
- 确认是否已生成测试数据
- 检查 localStorage 是否可用
- 运行 `viewCurrentData()` 查看数据

### 问题 3：筛选功能不生效
**检查项：**
- 检查 `activeFilter` 变量是否正确设置
- 确认日期比较逻辑正确
- 查看控制台是否有过滤错误

### 问题 4：详情无法打开
**检查项：**
- 检查数据结构是否完整
- 确认 `currentHistoryReport` 是否正确赋值
- 查看弹窗条件渲染逻辑
