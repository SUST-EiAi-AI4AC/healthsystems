# 快速启动指南

## 🚀 5分钟快速启动

### 第一步：检查数据库
```sql
-- 1. 连接到MySQL数据库
mysql -u root -p

-- 2. 选择数据库
USE healthsystem_test2;  -- 或者你的数据库名

-- 3. 检查必需的表是否存在
SHOW TABLES LIKE 'ai_chat%';
SHOW TABLES LIKE 'heal_diary';
SHOW TABLES LIKE 'user_health_info';

-- 4. 查看数据量
SELECT 'AI聊天会话' AS 表名, COUNT(*) AS 数量 FROM ai_chat_session
UNION ALL
SELECT 'AI聊天消息', COUNT(*) FROM ai_chat_message
UNION ALL
SELECT '日记记录', COUNT(*) FROM heal_diary
UNION ALL
SELECT '健康信息', COUNT(*) FROM user_health_info;
```

### 第二步：启动后端服务
```bash
# 方法1：使用Maven
cd healthsystem-backend6/healthsystem-backend
mvn spring-boot:run

# 方法2：使用IDE
# 直接运行 BackendApplication.java 主类
```

### 第三步：访问系统
1. 打开浏览器访问：http://localhost:8080
2. 使用管理员账号登录（默认账号请查看数据库）
3. 在左侧菜单中找到新增的三个模块：
   - AI聊天分析
   - 日记分析
   - 健康报告

## 📋 详细启动步骤

### 1. 环境准备

#### 必需软件
- ✅ JDK 8 或更高版本
- ✅ Maven 3.6+
- ✅ MySQL 8.0+
- ✅ IDE（推荐IntelliJ IDEA或Eclipse）

#### 检查环境
```bash
# 检查Java版本
java -version

# 检查Maven版本
mvn -version

# 检查MySQL版本
mysql --version
```

### 2. 数据库配置

#### 2.1 检查数据库连接配置
打开 `src/main/resources/application.yml`，确认数据库配置：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/healthsystem_test2?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password  # 修改为你的密码
```

#### 2.2 运行数据库检查脚本
```bash
# 在MySQL中运行
source 新增功能数据库检查.sql
```

### 3. 启动项目

#### 方法1：使用Maven命令行
```bash
# 进入项目目录
cd healthsystem-backend6/healthsystem-backend

# 清理并编译
mvn clean compile

# 启动项目
mvn spring-boot:run
```

#### 方法2：使用IDE
1. 用IDE打开项目
2. 找到 `BackendApplication.java`
3. 右键 -> Run 'BackendApplication'

#### 方法3：打包后运行
```bash
# 打包
mvn clean package

# 运行jar包
java -jar target/backend-0.0.1-SNAPSHOT.jar
```

### 4. 验证启动

#### 4.1 查看控制台日志
看到以下信息表示启动成功：
```
Started BackendApplication in X.XXX seconds
```

#### 4.2 访问测试
```bash
# 测试后端是否启动
curl http://localhost:8080

# 测试API文档
# 浏览器访问：http://localhost:8080/swagger-ui.html
```

### 5. 登录系统

#### 5.1 获取测试账号
```sql
-- 查询管理员账号
SELECT user_name, phone, role FROM user_info WHERE role IN ('doctor', 'admin');
```

#### 5.2 登录
1. 访问：http://localhost:8080
2. 输入用户名和密码（默认密码通常是：123456）
3. 登录成功后进入管理后台

### 6. 访问新功能

在左侧菜单中找到：
- 📊 **AI聊天分析** - 查看用户与AI助手的对话
- 📖 **日记分析** - 查看用户的情绪日记
- 💚 **健康报告** - 查看用户的综合健康报告

## 🔧 常见启动问题

### 问题1：端口被占用
```
Error: Port 8080 is already in use
```
**解决方案：**
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <进程ID> /F

# Linux/Mac
lsof -i :8080
kill -9 <进程ID>

# 或者修改端口
# 在application.yml中添加：
server:
  port: 8081
```

### 问题2：数据库连接失败
```
Error: Could not connect to database
```
**解决方案：**
1. 检查MySQL是否启动
2. 检查数据库名、用户名、密码是否正确
3. 检查防火墙设置
4. 检查MySQL是否允许远程连接

### 问题3：Maven依赖下载失败
```
Error: Could not resolve dependencies
```
**解决方案：**
```bash
# 清理Maven缓存
mvn clean

# 强制更新依赖
mvn clean install -U

# 或者配置国内Maven镜像（阿里云）
# 在 ~/.m2/settings.xml 中添加：
<mirror>
  <id>aliyun</id>
  <mirrorOf>central</mirrorOf>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

### 问题4：找不到Mapper
```
Error: Invalid bound statement (not found)
```
**解决方案：**
1. 检查Mapper接口的@Mapper注解
2. 检查XML文件路径是否正确
3. 清理并重新编译项目

### 问题5：权限验证失败
```
Error: 401 Unauthorized
```
**解决方案：**
1. 确认已登录
2. 检查Token是否过期
3. 清除浏览器缓存重新登录
